# pAnuncie

Plugin simples de anúncios para servidores Minecraft.

## Recursos

- Mensagens automáticas
- Configuração simples
- Compatível com Paper e Spigot
- Permissões personalizadas

## Comandos

| Comando | Descrição |
|----------|----------|
| `/panuncie reload` | Recarrega as configurações |

## Instalação

1. Baixe o arquivo `.jar`
2. Coloque na pasta `plugins`
3. Reinicie o servidor

## Compatibilidade

- Paper
- Spigot
- Purpur
